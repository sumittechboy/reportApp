### Troubleshooting Mobile Print

**Problem**: Print button does nothing
- **Solution**: Try refreshing the page (pull down and refresh)
- Make sure charts have finished loading

**Problem**: Charts don't appear in print
- **Solution**: Wait a few seconds after opening report before printing
- Check device has internet (charts are cloud-rendered)

**Problem**: Text is too small on printed page
- **Solution**: Increase print scale in print preview (usually default works)
- Try landscape orientation

**Problem**: PDF doesn't save on Android
- **Solution**: Check if app has storage permission
- Try a different browser (Chrome usually works best)
- Use "Save to Drive" or "Save to OneDrive" instead

**Problem**: Print option missing on iPhone
- **Solution**: Tap Share button (â¬† arrow) instead of Print
- Must have AirPrint printer or PDF option enabled
